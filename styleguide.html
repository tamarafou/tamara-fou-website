Style Guide

Mission

The mission of this website is to articulate my travel experiences.

Vision

I hope for this website to be a guide of any sort to those seeking advice, suggestions, etc. on travel activities for the destinations I've been to.

Target Audience

Any person keen to travel is my audience, but on top of naming classic/popular destinations, I will also try to write/post content that is unique and unlike my competitors, try to guide people to new places.

Logo

My logo is very simple. It is an image of a plane with my handwritten "Tams Tours" as the plane's logo. The colour scheme of this logo is also simple, it is meant to complement the colour scheme of the entirety of the website. The logo is to be placed on the top left hand corner, alongside the haeder of each page. Size will probably be 170x170.

















Colours

The colour scheme is intended to capture the essence of traveling via the colours we see in transit--skies, oceans, etc. The background colour will be Alice Blue (#F0F8FF), header fonts will be (#54a1ba), maps will be grey, and fonts will be black, some cases navy blue.



























Typography

The font will be set as the custom for html but will use Helvetica for headers. The font is not a major focus on my website because my website is intended to be more visual.
